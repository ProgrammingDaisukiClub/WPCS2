## Requirements

* Ruby: 2.4.0

* Node: 7.5.0

* npm: 4.1.2

## How to deploy
1. Make sure you have been installed gems: `rails`, `railties` and `bundler`

1. Install nodejs and npm

1. `bundler install`

1. `$ npm install`

 * build (for development)
  `$ npm run dev-build`
 * watch (for development)
  `$ npm run dev-watch`
 * build (for production)
  `$ npm run prod`
  
1. `rails server`
